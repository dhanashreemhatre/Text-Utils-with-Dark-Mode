# Text Utils with Dark Mode

This is a small project built with React that provides various text utility functions. It includes features such as converting text to uppercase or lowercase, checking readability, removing extra spaces, and copying text to the clipboard.

## Features

- **Convert Case**: Convert text to uppercase or lowercase.
- **Readability Check**: Evaluate the readability of the text using Flesch reading ease score.
- **Word Count**: Display the number of words and characters in the text.
- **Remove Extra Spaces**: Remove excess spaces from the text.
- **Copy to Clipboard**: Copy the text to the clipboard.

## Installation

To run this project locally, follow these steps:

1. Clone this repository to your local machine:

    ```bash
    git clone https://github.com/dhanashreemhatre/Text-Utils-with-Dark-Mode.git
    ```

2. Navigate to the project directory:

    ```bash
    cd Text-Utils-with-Dark-Mode
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

4. Start the development server:

    ```bash
    npm start
    ```

5. Open your browser and navigate to [http://localhost:3000](http://localhost:3000) to view the app.

## Usage

1. Enter the text in the textarea provided.
2. Utilize the buttons to perform various text manipulation operations.
3. Check the text summary for word count, character count, and estimated reading time.
4. Evaluate the readability of the text.
5. Enjoy the convenience of text utility functions!
